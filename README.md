# 🎵 Spotify Clone 🎧

A web-based clone of **Spotify**, built using **HTML**, **CSS**, and **JavaScript**. This project replicates the core functionalities and design of Spotify, allowing users to browse music, play songs, and manage playlists with a clean and responsive UI.

---

## 🔹 Features

- **🎶 Responsive UI**: Adapts to both desktop and mobile screens using Flexbox & Grid.
- **🎧 Music Player**: Play, pause, skip, and shuffle functionality for audio tracks.
- **📚 Playlist Management**: Create, edit, and delete custom playlists.
- **🔍 Search**: Basic search feature for discovering songs, albums, and artists.
- **📥 Static Data**: Uses hardcoded data for music content, with optional Spotify API integration for dynamic content.
- **🔧 Customizable**: Easily extendable to add user authentication, API integrations, and more.

---

## 🔹 Technologies Used

- **📝 HTML**: For the structure and semantics of the app.
- **🎨 CSS**: For styling and responsiveness with Flexbox, Grid, and Media Queries.
- **⚡ JavaScript**: For client-side interactivity, including music controls, search functionality, and playlist management.
- **🔌 External API (optional)**: Integrate Spotify’s Web API for dynamic music data like albums, artists, and tracks.

---

🔹 Running the Application
After following the setup instructions, you can run the app:

Open the index.html file in your browser.
The app should load, and you will be able to:
🎶 Play, pause, skip, and shuffle songs.
📚 Browse static playlists and albums.
🔍 Search for songs, albums, and artists (based on static data).
📝 Create, edit, and delete playlists.
